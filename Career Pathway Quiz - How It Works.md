# Career Pathway Quiz — How It Works

The Career Pathway Quiz is a short, three-question quiz designed to help users discover which construction careers align with their interests and skills. Users answer three multiple-choice questions, each with four options. Behind the scenes, each answer maps to a set of five skills. After all three questions are answered, the quiz compares the user's accumulated skills against the core skill profiles of 15 construction careers and presents the top three matches.

---

## How Scoring Works

### Step 1: Skill Accumulation

Each answer the user selects carries five hidden skill tags. Over three questions, the user accumulates up to 15 skill tags. Before scoring, any duplicate skills are removed so each skill is only counted once.

### Step 2: Matching Against Careers

Each of the 15 careers in the system has a defined list of core skills (ranging from 6 to 8 skills per career). The quiz checks how many of a career's core skills appear in the user's unique selected skills. The match percentage is calculated as:

> **(Number of matching skills ÷ Total core skills for that career) × 100**

For example, if a career has 8 core skills and 4 of them overlap with the user's selected skills, that career scores 50%. All skills are weighted equally — no single skill counts more than another.

### Step 3: Ranking and Results

All careers are sorted by their match percentage from highest to lowest. Ties are broken alphabetically. Only careers with a score above 0% are considered, and the top three are displayed to the user as result cards showing the career name, match percentage, and a brief description.

## Worked Example

Suppose a user selects the following answers:

1. **Question 1:** "Leading teams and projects" — Leadership, On-site leadership, Decision-making, Team coordination, Planning
2. **Question 2:** "Working with people" — Customer service, Negotiation, Teamwork, Communication, Problem-solving
3. **Question 3:** "Managing money and budgets" — Financial management, Budget management, Budgeting, Cost planning, Negotiation

After removing the duplicate (Negotiation appears twice), the user has 14 unique skills. The quiz then scores each career. For example:

- **Construction Manager** (8 core skills) — matches Leadership, Planning, Budgeting, Problem-solving, Communication, Teamwork = 6 out of 8 = **75%**
- **Project Manager** (7 core skills) — matches Leadership, Decision-making, Budget management, Communication, Negotiation = 5 out of 7 = **71%**
- **Quantity Surveyor** (8 core skills) — matches Cost planning, Problem-solving, Financial management, Negotiation = 4 out of 8 = **50%**

The top three careers by percentage would be displayed as the user's results.
